# AI Resume Analyzer

A GUI-based AI Resume Analyzer built using Python and Tkinter that evaluates resumes based on job-specific skills and provides intelligent feedback.

**Features**
- Analyze resume text based on selected job role
- Skill matching system
- Score calculation (0–100)
- Identifies missing skills
- Modern dark-themed GUI
- Progress bar visualization

**Tech Stack**
- Python
- Tkinter

**How It Works**
1. Select a job role
2. Paste your resume into the text box
3. Click on "Analyze Resume"
4. View your score, matched skills, and missing skills

**Run the Project**
```bash
python resume_analyzer_pro.py
